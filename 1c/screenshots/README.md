# Скриншоты и артефакты приёмки (реальная 1С)

Папка содержит доказательства работающего контура `1С → Kafka → PostgreSQL`
на реальной 1С:Предприятие 8.5.1.1302 (см. `../setup.md`).

## Программные артефакты (собраны автоматически)

| Файл | Что показывает |
|---|---|
| `01_ownership-forms.json` | Ответ HTTP-сервиса 1С `GET /ownership-forms` — 4 формы собственности |
| `02_counterparties.json` | Ответ HTTP-сервиса 1С `GET /counterparties` — 5 контрагентов |
| `03_changed-since-future-empty.json` | `?changed_since=<будущее>` → пустой массив (доказательство работы инкремента) |
| `04_postgresql-data.txt` | Итоговые данные в PostgreSQL после end-to-end + `/health` consumer |

## GUI-скриншоты 1С (снять вручную — см. ниже)

Для полноты комплекта сдачи по ТЗ (раздел 12) желательно приложить 3 скриншота
из GUI, которые нельзя получить программно:

1. **`gui_01_enterprise_catalogs.png`** — режим «1С:Предприятие», открыт справочник
   «Контрагенты» со списком из 5 записей (виден ИНН, форма собственности).
2. **`gui_02_iis_publication.png`** — Конфигуратор → «Публикация на веб-сервере»,
   видно имя `roshim`, каталог `C:\inetpub\wwwroot\roshim`, включённые галки.
3. **`gui_03_browser_json.png`** — браузер с открытым
   `http://localhost/roshim/hs/integration/counterparties` (JSON-ответ 1С).

> Программные `.json`/`.txt` артефакты уже доказывают работу контура; GUI-скриншоты
> дополняют комплект визуальным подтверждением настройки 1С.
