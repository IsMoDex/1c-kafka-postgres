# Доказательства работы и скриншоты (реальная 1С)

Папка содержит **программные артефакты (evidence)** работающего контура
`1С → Kafka → PostgreSQL` на реальной 1С:Предприятие 8.5.1.1302 (см. `../setup.md`),
а также место для опциональных GUI-скриншотов.

## Программные артефакты (входят в комплект, собраны автоматически)

| Файл | Что показывает |
|---|---|
| `01_ownership-forms.json` | Ответ HTTP-сервиса 1С `GET /ownership-forms` — 4 формы собственности |
| `02_counterparties.json` | Ответ HTTP-сервиса 1С `GET /counterparties` — 5 контрагентов |
| `03_changed-since-future-empty.json` | `?changed_since=<будущее>` → пустой массив (доказательство инкремента) |
| `04_postgresql-data.txt` | Итоговые данные в PostgreSQL после end-to-end + `/health` consumer (UTF-8) |

Этих артефактов достаточно, чтобы подтвердить работу контура на реальной 1С:
JSON-ответы 1С, семантику `changed_since` и конечное состояние PostgreSQL
(включая `deleted=true` у одного контрагента).

## GUI-скриншоты 1С (опционально, НЕ входят в текущий комплект)

Их нельзя получить программно; при желании дополнить визуальным подтверждением
настройки 1С положите сюда PNG (снять через Win+Shift+S):

1. `gui_01_enterprise_catalogs.png` — режим «1С:Предприятие», справочник
   «Контрагенты» со списком из 5 записей (виден ИНН, форма собственности).
2. `gui_02_iis_publication.png` — Конфигуратор → «Публикация на веб-сервере»
   (имя `roshim`, каталог `C:\inetpub\wwwroot\roshim`).
3. `gui_03_browser_json.png` — браузер с открытым
   `http://localhost/roshim/hs/integration/counterparties`.

> Статус: программные артефакты присутствуют и самодостаточны. GUI-скриншоты —
> необязательное дополнение, добавляются вручную.
